☕ Java 基础语法

学习日期：2025-11-15 难度：★★☆☆☆

Java 是一种面向对象的编程语言，具有跨平台性，通过 JVM（Java 虚拟机）实现一次编写多处运行。

核心知识点：

  • 基本数据类型：byte(1字节)、short(2字节)、int(4字节)、long(8字节)、float(4字节)、double(8字节)、char(2字节)、boolean
  • 引用类型：类、接口、数组，默认值为null
  • 访问修饰符：public > protected > default > private
  • 封装：通过private修饰符隐藏内部实现，提供public getter/setter方法访问
  • 继承：使用extends关键字，Java单继承多实现特性
  • 多态：通过方法重写和接口实现体现，运行时动态绑定

代码示例：

public class HelloWorld {
    public static void main(String[] args) {
        // 基本输出语句
        System.out.println("Hello, World!");
        
        // 变量定义与使用
        int a = 10;
        String name = "Java";
        System.out.println("变量a的值：" + a);
        System.out.println("语言名称：" + name);
    }
}

练习题：

编写一个Java程序，实现两个整数的加减乘除运算，并处理除数为0的异常。

public class Calculator {
    public static void main(String[] args) {
        int num1 = 10;
        int num2 = 0;
        
        try {
            System.out.println("加法：" + (num1 + num2));
            System.out.println("减法：" + (num1 - num2));
            System.out.println("乘法：" + (num1 * num2));
            System.out.println("除法：" + (num1 / num2));
        } catch (ArithmeticException e) {
            System.out.println("除法错误：除数不能为0");
        }
    }
}

🐍 Python 基础语法

学习日期：2025-11-15 难度：★☆☆☆☆

Python 是一种简洁易学的高级编程语言，以可读性强著称，采用缩进代替大括号来定义代码块。

核心知识点：

  • 数据结构：列表(list)、元组(tuple)、字典(dict)、集合(set)

  • 模块系统：使用import导入模块，from...import...选择性导入

代码示例：

# 基本输出
print("Hello, World!")

# 变量定义
a = 10
name = "Python"
print(f"变量a的值：{a}")
print(f"语言名称：{name}")

# 列表操作
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"水果：{fruit}")

练习题：

编写一个Python函数，计算列表中所有偶数的和。

def sum_even_numbers(numbers):
    total = 0
    for num in numbers:
        if num % 2 == 0:
            total += num
    return total

# 测试
nums = [1, 2, 3, 4, 5, 6]
print(f"偶数和：{sum_even_numbers(nums)}")  # 输出：12

function createColorButton() {
    var button = document.createElement('button');
    button.textContent = "切换背景色";
    button.style.padding = "10px 20px";
    button.style.cursor = "pointer";
    
    var colors = ["#ff0000", "#00ff00", "#0000ff", "#ffff00", "#ff00ff"];
    var colorIndex = 0;
    
    button.addEventListener('click', () => {
        document.body.style.backgroundColor = colors[colorIndex];
        colorIndex = (colorIndex + 1) % colors.length;
    });
    
    document.body.appendChild(button);
}

// 调用函数创建按钮
createColorButton();

核心知识点：

  • 聚合函数：COUNT()、SUM()、AVG()、MAX()、MIN()
  • 连接查询：INNER JOIN、LEFT JOIN、RIGHT JOIN

代码示例：

-- 创建数据库
CREATE DATABASE IF NOT EXISTS myblog;
USE myblog;

-- 创建用户表
CREATE TABLE IF NOT EXISTS users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 插入数据
INSERT INTO users (username, password, email)
VALUES ('admin', '123456', 'admin@example.com'),
       ('user1', '654321', 'user1@example.com');

-- 查询数据
SELECT id, username, email FROM users WHERE id > 0;

-- 更新数据
UPDATE users SET password = 'newpassword' WHERE username = 'admin';

-- 删除数据
DELETE FROM users WHERE id = 2;

🔧 C 语言基础语法

学习日期：2025-11-16 难度：★★★☆☆

C语言是一种面向过程的编程语言，由丹尼斯·里奇于1972年在贝尔实验室开发。它是许多现代编程语言的基础， 具有高效、灵活和接近硬件的特点，广泛应用于系统编程、嵌入式开发等领域。

核心知识点：

  • 基本数据类型：char(1字节)、int(4字节)、float(4字节)、double(8字节)
  • 指针：变量的内存地址，支持指针运算和多级指针
  • 数组：相同类型元素的集合，通过下标访问，不进行边界检查
  • 函数：程序的基本模块，支持递归调用和函数指针
  • 预处理指令：#include、#define、#ifdef等，在编译前处理
  • 内存管理：手动分配(malloc)和释放(free)动态内存

代码示例：

#include <stdio.h>
#include <stdlib.h>

// 计算阶乘的函数
int factorial(int n) {
    if (n <= 1)
        return 1;
    else
        return n * factorial(n - 1);
}

int main() {
    // 基本输出
    printf("Hello, World!\n");
    
    // 变量定义与使用
    int a = 10;
    char name[] = "C Language";
    printf("变量a的值：%d\n", a);
    printf("语言名称：%s\n", name);
    
    // 调用函数
    int num = 5;
    printf("%d的阶乘是：%d\n", num, factorial(num));
    
    return 0;
}

练习题：

编写一个C程序，使用指针实现两个整数的交换。

#include <stdio.h>

// 使用指针交换两个整数的值
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    
    printf("交换前：x = %d, y = %d\n", x, y);
    swap(&x, &y);
    printf("交换后：x = %d, y = %d\n", x, y);
    
    return 0;
}

➕ C++ 基础语法

学习日期：2025-11-16 难度：★★★★☆

C++是在C语言基础上发展而来的编程语言，保留了C语言的高效性和灵活性，同时引入了面向对象编程的特性。 它广泛应用于系统开发、游戏引擎、高性能计算等领域，是一种功能强大且复杂的编程语言。

核心知识点：

  • 面向对象：类(class)、对象、封装、继承、多态
  • 标准模板库(STL)：vector、string、map等容器和算法
  • 函数重载：同一作用域内同名函数，参数列表不同
  • 运算符重载：自定义运算符对类对象的操作
  • 引用：变量的别名，可作为函数参数和返回值
  • 异常处理：try-catch-finally机制处理运行时错误

代码示例：

#include <iostream>
#include <string>
using namespace std;

// 定义一个简单的类
class Person {
private:
    string name;
    int age;
    
public:
    // 构造函数
    Person(string n, int a) : name(n), age(a) {}
    
    // 成员函数
    void introduce() {
        cout << "我叫" << name << "，今年" << age << "岁。" << endl;
    }
    
    // 年龄增长函数
    void growOld() {
        age++;
    }
};

int main() {
    // 基本输出
    cout << "Hello, World!" << endl;
    
    // 创建对象
    Person person("张三", 20);
    person.introduce();
    
    // 调用对象方法
    person.growOld();
    person.introduce();
    
    return 0;
}

练习题：

编写一个C++程序，定义一个Shape基类和Circle、Rectangle派生类，实现多态特性计算面积。

#include <iostream>
using namespace std;

// 形状基类
class Shape {
public:
    // 纯虚函数，计算面积
    virtual double calculateArea() = 0;
    // 虚析构函数
    virtual ~Shape() {}
};

// 圆形类
class Circle : public Shape {
private:
    double radius;
    
public:
    Circle(double r) : radius(r) {}
    
    // 重写计算面积方法
    double calculateArea() override {
        return 3.14159 * radius * radius;
    }
};

// 矩形类
class Rectangle : public Shape {
private:
    double length;
    double width;
    
public:
    Rectangle(double l, double w) : length(l), width(w) {}
    
    // 重写计算面积方法
    double calculateArea() override {
        return length * width;
    }
};

// 多态使用示例
void printArea(Shape* shape) {
    cout << "面积是：" << shape->calculateArea() << endl;
}

int main() {
    Shape* circle = new Circle(5.0);
    Shape* rectangle = new Rectangle(4.0, 6.0);
    
    printArea(circle);    // 输出圆形面积
    printArea(rectangle); // 输出矩形面积
    
    delete circle;
    delete rectangle;
    
    return 0;
}

每一行代码都是思想的具象化，写得越多越明白：好的代码和优质的人生一样，都需要清晰的结构、适当的注释和不断的重构。

生活态度：

  • 不必苛求完美，接纳生活中的不圆满，才是真实的人生
  • 珍惜日常的小确幸，一杯热茶、一本好书、一次真诚的交谈，都是生活的馈赠
  • 保持对世界的好奇心，像孩子一样观察生活，会发现平凡中的不凡
  • 学会与自己相处，独处时的思考比喧嚣时的附和更有价值
  • 与其纠结无法改变的过去，不如专注可塑造的未来，生活的操作系统需要定期更新，但不必频繁重启
  • 给生活留白，就像代码中的注释行，适当的空白能让整体结构更清晰，也让心灵有呼吸的空间
  • 遇到困境时，像调试程序一样分步拆解问题，复杂问题往往有简单的突破口
  • 保持终身学习的热情，如同持续迭代的软件版本，每一次更新都让自己更完善
  • 给生活设定"版本迭代"计划，每月学习一项新技能，每年完成一次深度旅行，让成长有迹可循
  • 像优化代码性能一样精简生活，定期"清理内存"——断舍离无用物品，告别消耗型社交
  • 接纳"调试期"的不完美，刚接触新领域时允许自己笨拙，就像编译程序总会遇到几次报错
  • 培养"离线模式"的能力，每天留一段不看屏幕的时间，让思绪像未被占用的进程一样自由流动
  • 把困难当作"压力测试"，顺利时的成长是线性的，突破瓶颈后的进步才是指数级的

社会感触：

  • 信息爆炸的时代，更需要培养独立思考能力，不被流量裹挟，不随波逐流
  • 快节奏的生活中，人与人之间的真诚连接变得尤为珍贵
  • 社会的进步不仅需要科技的发展，更需要人文关怀的温度
  • 每个人都是社会的微小齿轮，做好自己分内之事，就是在为世界添砖加瓦
  • 数字化时代，人与人的物理距离被拉近，但心与心的距离需要更多真诚互动来维系
  • 技术发展应该服务于人的幸福，而不是让人成为算法的奴隶，保持人性的温度比追求效率更重要
  • 社会就像一个大型分布式系统，每个节点的健康运行都关系到整体稳定，责任需要共同承担
  • 在快速变化的时代，保持内心的锚点很重要，那些经过时间沉淀的价值观如同代码的核心架构，支撑着我们应对各种变化
  • 数字鸿沟正在悄然拉大，有人用技术提升效率，有人被算法困在信息茧房，技术普惠需要更多人的努力
  • 快节奏社会里，"慢慢来"反而成为稀缺能力——深度阅读、专注创作、耐心陪伴，这些慢动作正在变得珍贵
  • 社交媒体让表达成本降低，但高质量的倾听却越来越少，真正的沟通需要双向的信息交换而非单向输出
  • 职业无高低，但社会对"有用"的定义正在变得狭窄，那些守护平凡日常的工作同样值得尊重
  • 环境问题如同未处理的内存泄漏，短期内不影响运行，长期却会导致系统崩溃，需要每个人编写"环保补丁"
